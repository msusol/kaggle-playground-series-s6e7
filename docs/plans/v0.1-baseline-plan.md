# v0.1 - Baseline Plan

Goal: lock the submission pipeline and stratified CV harness with a cheap
LightGBM/XGBoost multiclass model, per Rung 0-1 of the implementation plan.

## Steps
1. `zsh scripts/download_data.sh`.
2. EDA: confirm target balance (85.9% / 8.4% / 5.8%) and per-column missingness
   (1-12%) match between train/test; check for leakage-prone columns.
3. LightGBM multiclass, native categorical dtype for the 6 categorical columns,
   `class_weight='balanced'`, 5-fold **stratified** CV.
4. Predict test, write `submission.csv` (`id,health_condition`), verify header +
   row count (295,753 rows) against `sample_submission.csv`.
5. Submit via `kaggle competitions submit -c playground-series-s6e7 -f submission.csv -m "v0.1 baseline"`.

## Acceptance
- CV (balanced accuracy) beats the ~0.333 majority-class floor.
- `submission.csv` matches `sample_submission.csv` format exactly.
- Record CV/LB in leaderboard.md.
